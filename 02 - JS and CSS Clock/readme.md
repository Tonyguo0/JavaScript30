# JS and CSS Clock

This is a simple clock made with JavaScript and CSS. It displays the current time in hours, minutes, and seconds, and updates every second.

## Learning points

- We have learnt transform-origin as part of css it off-sets the origin of the transformation, such as rotation, to a different point. In this case, we set the transform-origin to the center of the clock hands, so that they rotate around that point. aka the <b>right side of the clock hand/ center of the clock</b> is the center of rotation.
